**Utilisation de base**

**Push to Talk (PTT)**

1. Cliquez sur "Push to Talk"
2. Appuyez sur **Ctrl+Espace** et commencez à parler
3. Relâchez - du texte apparaîtra à l'emplacement du curseur

Idéal pour les phrases courtes. C'est le mode par défaut.

**Détection d'activité vocale (VAD)**

1. Cliquez sur les trois points (...) pour ouvrir le panneau de paramètres
2. Réglez le déclencheur sur **VAD**
3. Commencez à parler - OFFTYPE détectera automatiquement les phrases et tapera chacune lorsqu'elle sera terminée

Aucune touche n'est nécessaire. Idéal pour les entrées vocales plus longues.

**Comment savoir si VAD fonctionne ?**

Observez la couleur du bouton Push to Talk :
- **Vert clignotant (lumière respirante)** : La détection de parole fonctionne normalement
- **Vert constant, sans clignotement** : Le moteur VAD est en cours d'exécution mais ne détecte pas la parole. Essayez de redémarrer OFFTYPE. Si le problème persiste, vérifiez que le modèle ASR est téléchargé dans les préférences.

**Basculement des modes**

Utilisez la liste déroulante **Processus** dans le panneau étendu :

| Mode | Résultat |
| --- | --- |
| Entrée | Parlez → texte (par défaut) |
| Traduction | Parlez chinois → texte traduit |
| Mise en forme | Parlez → texte réécrit dans votre style choisi |

**Panneau de paramètres**

Cliquez sur l'icône engrenage pour ouvrir. Quatre options sont disponibles :

1. **Déclencheur** : PTT (push-to-talk) ou VAD (écoute continue)
2. **Processus** : Entrée / Traduction / Mise en forme
3. **Cible** : Langue de sortie (mode traduction)

  - Cliquez avec le bouton droit sur l'icône de la barre d'état système → Paramètres → Dispositif audio, sélectionnez le microphone correct.

  - La première exécution après le lancement peut être plus lente pendant que le modèle se charge en mémoire vive (RAM).
  - Passez à SenseVoice-Small (plus rapide) si vous n'avez pas besoin de la prise en charge des dialectes.
  - Désactivez la traduction si vous n'avez besoin que de la transcription.

  - Vérifiez votre connexion Internet. Les téléchargements de modèles nécessitent une connexion unique à Hugging Face ou ModelScope.

  - Ollama est-il activé dans les préférences mais ne fonctionne pas ?
  - Le modèle local LLM est-il téléchargé ?

- **Téléchargement lent:**
  - SenseVoice télécharge à partir de HuggingFace / ModelScope. Dans la plupart des régions, ModelScope est automatiquement pris en charge.
  - Qwen3-ASR télécharge à partir des releases GitHub. OFFTYPE sonde les miroirs disponibles et utilise le plus rapide.

  - Attendez 5 secondes — le serveur a besoin de temps pour démarrer.
  - Ouvrez le Gestionnaire des tâches pour vérifier si `speech-server.exe` est en cours d'exécution.
  - Si une version précédente était installée, tuez les processus anciens en premier: `taskkill /F /IM speech-server.exe`.

  - Menu Démarrer → Paramètres → Applications → Applications installées → Recherchez OFFTYPE → Désinstaller.
  - Les fichiers de modèle sont stockés dans `%APPDATA%\OFFTYPE\models\`. Vous devrez supprimer ce dossier manuellement après la désinstallation.

**FAQ:**

- **OFFTYPE envoie-t-il ma voix quelque part ?**
Non. La reconnaissance vocale, la ponctuation et la traduction se produisent toutes sur votre machine. Voir [Confidentialité](/confidentialite/) pour la preuve de Wireshark.

- **OFFTYPE fonctionne-t-il hors ligne ?**
Oui, après le premier téléchargement du modèle. L'utilisation ultérieure est entièrement hors ligne.

- **Puis-je l'utiliser sur macOS ou Linux ?**
Pas pour le moment. OFFTYPE est uniquement pour Windows. macOS et Linux ne sont pas prévus dans un avenir proche.

- **Y a-t-il une version payante ?**
OFFTYPE est actuellement gratuit. Il n'y a pas d'abonnement, pas de compte, pas de paiement.

- ** Puis-je l'exécuter sur un réseau d'entreprise ?**
Oui. OFFTYPE ne nécessite qu'un accès réseau pour le téléchargement initial du modèle et les mises à jour éventuelles. Après cela, il fonctionne entièrement hors ligne. Si votre pare-feu bloque Hugging Face, configurez le miroir ModelScope.

- **Comment le désinstaller ?**
Menu Démarrer → Paramètres → Applications → Applications installées → Recherchez OFFTYPE → Désinstaller. Pour supprimer toutes les données, supprimez manuellement `%APPDATA%\OFFTYPE\`.
